## GIG Construction API

Hybrid content and inquiry management app built on Laravel 12 with a focus on security, media handling, and multilingual support.

### Tech stack
- Backend: PHP 8.2, Laravel 12, Fortify (auth), Sanctum (API tokens), Spatie Media Library, Spatie Translation Loader, Symfony Mailer (Mailgun), AWS S3 storage driver, Predis/Redis for cache and queues.
- Frontend: Vite 7, Tailwind CSS 4, Axios, Laravel Vite Plugin.
- Tooling: Pest for tests, Laravel Pint for style, Laravel Boost/Pail for local diagnostics, Docker (MySQL) optional.

### Requirements
- PHP 8.2+, Composer 2.x
- Node.js 20+ and npm
- MySQL/MariaDB and Redis (or compatible cache/queue backend)
